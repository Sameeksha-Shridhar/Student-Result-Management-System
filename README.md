# Student Result Management System

A sleek and user-friendly Python GUI application designed to manage, track, and organize student academic records efficiently.

## 🌟 Features
* **Interactive GUI:** Built with Python's Tkinter/CustomTkinter framework for a smooth user experience.
* **Database Management:** Uses SQLite to securely store, update, and retrieve student results locally.
* **CRUD Operations:** Easily add new student records, view existing data, modify scores, and delete entries.

## 🛠️ Built With
* **Language:** Python 3.x
* **GUI Framework:** Tkinter
* **Database:** SQLite3
